## CLI vs GUI

- **GUI**: Main interface that uses icons

## Linux Family

## File Explorer (Nautilus) 
- Open for GUI: File Manager (blue folder)
- Open for CLI:
  ```bash
  nautilus
  ```
- Change display: Ctrl + 1 / Ctrl + 2
- Show hidden files: Ctrl + H
- Open magnifying glass (search): Ctrl + F
- Add path to search: Ctrl + L

## Text Editor (gedit)
- Open for GUI: Text Editor (gedit)
- Open for CLI:
  ```bash
  gedit
  ```
